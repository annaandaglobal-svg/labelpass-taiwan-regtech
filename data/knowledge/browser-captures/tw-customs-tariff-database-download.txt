Official CPT Single Window / Customs Administration page GC453 Tariff Database Download. The page is a browser-oriented official download page for Taiwan tariff and CCC-code reference data. It identifies the CCC CODE last modification date and presents official download options for tariff and commodity-code work. Listed resources include Tariff Schedule files in spreadsheet, PDF, and open-document formats; CCC Code and Description of Goods files in text and PDF formats; chapter-by-chapter tariff materials; Application for Import Goods Advance Tariff Rulings files in document, PDF, and open-document formats; and Tariff No. and Description of Goods files in text and PDF formats. The page is relevant when a product team needs to confirm the Taiwan customs statistical classification, compare HS candidates, download the current tariff schedule, or prepare an advance classification ruling before import. The tariff schedule and CCC goods-description files support screening for duty rate, import/export controls, TFDA routing, BSMI routing, SHTC routing, and other agency rules that depend on commodity classification. Advance tariff ruling materials are important when product composition, use, kit status, or mixed-material goods make the final CCC code uncertain.
