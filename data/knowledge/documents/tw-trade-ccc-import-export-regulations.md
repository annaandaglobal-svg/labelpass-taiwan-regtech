# Import and Export Regulations by CCC Code

Source: https://fbfh.trade.gov.tw/fh/indexE.jsp

## Extract

Official Taiwan International Trade Administration/MOEA database page for Classification of Commodities and Regulations. The English interface is titled Import and Export Regulations and supports searching by commodity code, commodity code range, or imported file. It accepts HS code inputs from 2 to 6 digits and CCC code inputs under the Standard Classification of Commodities of the Republic of China up to 10 digits, with examples including 01, 2001, 740729, 0203120000, and 2207101010. The page also supports commodity-description keyword search, such as fish, bovine animals, or chemical, and regulation-code search for import or export regulation types. The form refers to Import Regulation codes and Export Regulation codes, with examples including numeric codes and alphanumeric codes such as 111, 401, 272, C01, and MW0. The page provides download areas for import/export regulation information and an Importation of Mainland China products section. It explains that where EX is appended after a CCC number, only goods conforming to the corresponding Chinese and English commodity description are opened for import. It also notes that where Mainland China product import special regulation codes such as MXX or NXX are listed, the goods must satisfy both general import regulations and those special provisions before import is allowed. The user workflow is to identify a commodity by HS or CCC number, confirm whether an import regulation code, export regulation code, permit, inspection, or agency restriction appears, and then map the result back to the responsible agency and shipment documents. This makes the page a primary routing source before preparing invoices, packing lists, origin evidence, TFDA documents, SHTC checks, or customs entries.
